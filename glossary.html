Glossário HTML

html
Define um documento HTML. :)
head
É um container para metadados, título da página, links e etc.
body
É o corpo do documento, onde deve estar todo o conteúdo da página.
títulos (H1, H2, H3)
Define o texto como título, subtítulo e afins. Cada tag representa um tamanho, sendo h1 a principal, de título mesmo. :)
label
Define o título de um formulário, mostrando para o usuário o conteúdo que ele deve inserir no form.
input
Utilizado para receber dados do usuário.
button
Adiciona um botão clicável na página.
